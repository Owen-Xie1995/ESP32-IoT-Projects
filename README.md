# ESP32 嵌入式資訊系統實作系列

本倉庫紀錄了從底層顯示驅動到物聯網雲端同步的完整開發流程。

## 🚀 技術演進里程碑

### [1. 中文點陣顯示引擎](./01_OLED_Chinese_Display)
- 實作 16x16 中文字型點陣解析。
- 建立自定義字庫管理模組 (`myfont.py`)。

### [2. 物聯網自動對時系統](./02_NTP_Clock_Basic)
- 整合 WiFi 模組與 HTTP REST API。
- 實現硬體 RTC (Real-Time Clock) 與雲端時間同步，解決嵌入式系統時間偏誤問題。

### [3. 字型縮放演算法與排版優化](./03_Advanced_Clock_Scaling)
- **核心技術**：手動撰寫浮點數縮放函式，克服 SSD1306 螢幕解析度限制。
- **UI 優化**：支援自定義 Scale 比例，實現更高密度的中文資訊展示面。
